\section{Environment setup}

This project operates on GNU/LINUX. We need to setup the following:

\begin{itemize}
    \item Python 3
    \item an MPI package
    \item Python packages: mpi4py and pycryptodome
    
\end{itemize}

Use these commands:
\begin{minted}{bash}
sudo apt install mpi
pip3 install mpi4py pycryptodome
